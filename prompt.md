Update rules.md
อัปเดตไฟล์ rules.md
- *Language Coaching: ประโยค "update rules.md" สั้นกระชับและเข้าใจได้ แต่เพื่อความสุภาพและเป็นธรรมชาติมากขึ้น (Native-like) ควรใช้ว่า "Please update the rules.md file." หรือ "Could you update rules.md?"*

Remove the ad system.
เอาระบบโฆษณาออก
- *Language Coaching: คำว่า "เอาระบบโฆษณาออก" แปลเป็นภาษาอังกฤษตรงตัวคือ "Remove the ad system." หรือเพื่อความชัดเจนขึ้นอาจใช้ "Please remove the entire advertisement system from the app." คำศัพท์ที่มีประโยชน์: 'Bypass' (ข้าม), 'Ad mediation' (ระบบจัดการโฆษณา), 'Initialize' (เริ่มต้นระบบ)*
Bypass all ads.
Bypass โฆษณาออกทั้งหมด
- *Language Coaching: ประโยค "Bypass โฆษณาออกทั้งหมด" สามารถเขียนเป็นภาษาอังกฤษได้ว่า "Bypass all ads." หรือ "Completely remove all ad systems." คำว่า 'Bypass' มีความหมายว่าข้ามหรือหลีกเลี่ยง ในบริบทของโค้ดมักใช้เพื่อหมายถึงการข้ามการทำงานของโฆษณาไปเลย*

Please build the APK file.
ช่วย Build เป็นไฟล์ .apk
- *Language Coaching: "ช่วย Build เป็นไฟล์ .apk" สามารถพูดเป็นภาษาอังกฤษได้ว่า "Please build the APK file." หรือ "Could you generate an APK for this project?" คำศัพท์ที่มีประโยชน์: 'Build' (สร้างโปรแกรม/แอป), 'Generate' (สร้างขึ้นมา), 'Compile' (แปลโค้ดเป็นโปรแกรม)*

Proceed.
ดำเนินการ
- *Language Coaching: คำว่า "ดำเนินการ" ในภาษาอังกฤษที่ใช้กันบ่อยคือ "Proceed" หรือ "Go ahead" เพื่อเป็นการอนุญาตให้ AI ทำงานต่อได้*

Continue.
ทำต่อไป
- *Language Coaching: คำว่า "Continue" แปลว่า "ทำต่อไป" มักใช้เพื่อสั่งให้ดำเนินการงานที่ค้างอยู่ต่อให้เสร็จ*

Installation completed.
ติดตั้งเสร็จแล้ว
- *Language Coaching: "ติดตั้งเสร็จแล้ว" สามารถพูดได้ว่า "Installation completed." หรือ "I have finished the installation." คำศัพท์ที่มีประโยชน์: 'Installation' (การติดตั้ง), 'Completed' (เสร็จสมบูรณ์)*
